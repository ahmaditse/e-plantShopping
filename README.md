# Paradise Nursery

Paradise Nursery is a modern React-based e-commerce web application for selling houseplants. The project features a beautiful landing page, a product listing page, and a shopping cart powered by Redux.

## Project Overview

This project is a full-stack-inspired frontend application built with React and Redux. Users can browse houseplants, add them to a cart, and navigate between the Home, Plants, and Cart pages.

---

## Features

- **Landing Page** with full-page background and split layout:
  - Left 2/3: descriptive text (Lorem Ipsum placeholder)
  - Right 1/3: title and "Get Started" button
  - Navbar with clickable logo and green background

- **Product Listing Page**:
  - At least six houseplants per category
  - Categorized into Indoor Plants, Outdoor Plants, etc.
  - Add to Cart button disables after adding
  - Plant images clickable as links

- **Shopping Cart**:
  - Displays added plants with quantity and total
  - Increment and decrement quantity
  - Delete individual items
  - Checkout button shows "Coming Soon"

- **Responsive Design** for desktop and mobile

- **State Management** using Redux Toolkit
